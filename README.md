## English
## 1. What is Spring Security?
Spring Security is a framework that handles the security of Spring-based applications. Spring Security provides many features related to security, such as authentication, authorization, access control, and attack defense. Spring Security is provided as a module of Spring.

## 2. Features of Spring Security
* Spring Security is a framework that handles the security of Spring-based applications.
* Spring Security provides many features related to security, such as authentication, authorization, access control, and attack defense.
* Spring Security is provided as a module of Spring.
* Spring Security provides many features, but it is easy to configure and flexible.
* Spring Security supports various authorization methods.
* Spring Security provides various attack defense features.

## 3. Structure of Spring Security
Spring Security is divided into three main areas.
* Authentication: The process of verifying who the user is.
* Authorization: The process of verifying whether the user can perform a specific task.
* Access Control: The process of verifying whether the user can access a specific resource.

## 4. How to configure Spring Security
Spring Security is divided into three types depending on the configuration method.
* XML configuration method
* Java configuration method
* Java configuration + XML configuration method

## 5. Main features of Spring Security
Spring Security provides various features.
* Authentication: The process of verifying who the user is.
* Authorization: The process of verifying whether the user can perform a specific task.
* Access Control: The process of verifying whether the user can access a specific resource.
* Attack Defense: Provides defense features against various attacks.

## 6. Authentication methods of Spring Security
Spring Security supports various authentication methods.
* Form-based authentication
* HTTP basic authentication
* HTTP digest authentication
* LDAP authentication
* X.509 authentication
* CAS authentication
* OAuth authentication

## 7. Authorization methods of Spring Security
Spring Security supports various authorization methods.
* Role-based authorization
* URL-based authorization
* Method-based authorization
* Expression-based authorization

## 8. Attack defense features of Spring Security
Spring Security provides defense features against various attacks.
* CSRF (Cross-Site Request Forgery) defense
* XSS (Cross-Site Scripting) defense
* Clickjacking defense
* SQL Injection defense
* Session Fixation defense
* URL security
* File upload security
